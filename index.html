Anmäl här om någon bryter mot din licens/upphovsrätt!

Har du hittat en användning av ett verk du har laddat upp till Wikimedia Commons där licensen inte anges ordentligt?
Då kan du använda nedanstående verktyg för att snabbt sätta ihop ett e-post till illdådaren.












*

Kopiera text och formatering genom att trycka ctrl + c eller ⌘ + c (du behöver inte markera texten).

Notera att all formatering tappas vid export genom ovanstående knapp.
För att behålla formateringen bör du istället klistra in ovanstående i din e-postklient.
